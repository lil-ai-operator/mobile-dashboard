## 3. International Expansion: Ireland & EU

### Why Ireland First?

| Factor | Rationale |
|--------|-----------|
| **Language** | English-speaking — no translation needed |
| **Legal/Regulatory** | EU member, GDPR compliant, familiar legal system |
| **Timezone** | GMT (same as UK) — easy support overlap |
| **Tech Ecosystem** | Dublin = "Silicon Valley of Europe" — talent, investors |
| **Tax** | 12.5% corporate tax — startup-friendly |
| **Market Size** | 5M population, high digital adoption |

### Expansion Roadmap: Ireland → EU

#### Phase 1: Ireland (Months 1-3)
- [ ] Set up Irish business entity (or use UK entity with Irish branch)
- [ ] Add EUR pricing tier
- [ ] Localise: GBP → EUR, add Irish VAT handling
- [ ] Partner with 2-3 Irish business associations
- [ ] Target: 20 pilot businesses in Dublin/Cork

#### Phase 2: EU Core (Months 4-9)
- [ ] Expand to: Netherlands, Germany, France, Spain
- [ ] GDPR compliance audit (already largely covered)
- [ ] Multi-language support: NL, DE, FR, ES
- [ ] Local payment methods: iDEAL, SEPA, Bancontact
- [ ] Partner with EU-focused salon/consultant networks

#### Phase 3: Full EU (Months 10-18)
- [ ] Complete EU-27 coverage
- [ ] Local phone numbers (本地号码) in each market
- [ ] Region-specific industry templates
- [ ] VATMOSS compliance for digital services

### Key Considerations

- **GDPR**: AutoBookr already handles personal data; ensure processing agreements are EU-compliant
- **Payments**: Stripe already supports EUR; add local invoicing
- **Support**: Hire/contract part-time EU support (Irish or Netherlands-based)
- **Marketing**: Target English-speaking EU businesses first, then localise

## 4. White-Label Concept

### What Is White-Label AutoBookr?

Sell AutoBookr under your partners' brand. They resell to their clients as "their" product.

### Use Cases

| Partner Type | Example |
|--------------|---------|
| **Digital Agencies** | Offer "AI Booking" as part of their web design package |
| **Marketing Agencies** | Resell to clients as a lead-gen tool |
| **Business Consultants** | Bundle with operational efficiency consulting |
| **SaaS Resellers** | Add to existing portfolio of SMB tools |
| **Franchise Networks** | Roll out to all franchise locations under their brand |

### White-Label Features

| Feature | Description |
|---------|-------------|
| **Custom Domain** | booking.[partnerbrand].com |
| **Branded UI** | Logo, colors, custom footer text |
| **Partner Dashboard** | See all their clients' usage |
| **Client Management** | Partner adds/removes end-users |
| **Custom Pricing** | Partner sets their own pricing |
| **API Access** | Deep integrations with partner's stack |

### Pricing Model

| Tier | Monthly Fee | Max Clients | Features |
|------|-------------|-------------|----------|
| **Starter** | £99/mo | 10 | Basic white-label |
| **Growth** | £249/mo | 50 | Full branding + API |
| **Enterprise** | £499/mo | Unlimited | Multi-user + priority support |

### Implementation Path

1. **Multi-tenant architecture** — Separate data per partner/client
2. **Custom theming engine** — CSS variables for brand colors
3. **Partner portal** — Dashboard to manage end-clients
4. **Reseller billing** — Partner pays AutoBookr, they bill clients
5. **API-first** — Enable deep integrations
