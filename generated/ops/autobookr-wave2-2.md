## 3. Word-of-Mouth Referral Program

### Program Name: "Fren Recommendations"

### How It Works

```
┌──────────────────────────────────────────┐
│  REFER A MATE, BOTH GET £20 OFF         │
│     → No limits, ongoing program         │
└──────────────────────────────────────────┘
```

### Mechanics

| Action | Reward (Referrer) | Reward (Referee) |
|--------|-------------------|------------------|
| Successful referral | £20 credit | £20 off first month |
| 5+ referrals in a month | £50 bonus | - |
| Year-end top referrer | £200 + plaque | - |

### Referral Channels

1. **In-app referral** (post-checkout)
   - "Know a mate in the trades? Share your link!"

2. **Email signature**
   - Auto-generated signature with link

3. **Social share**
   - Pre-made graphics for Facebook, Instagram, LinkedIn

4. **Printed cards**
   - Tuck into packaging/packs

5. **Trade events**
   - Business cards with QR to referral page

### Tracking & Attribution
- Unique referral link per user
- Cookie-based tracking (90-day window)
- Dashboard showing: referrals, conversions, earnings

### Anti-Abuse Rules
- Self-referrals blocked
- Same business = 1 referral max
- Suspicious patterns = manual review

### Promotion Ideas
- "Fren of the Month" spotlight on socials
- Quarterly prize draw (tool voucher) for active referrers
- Partnership with tool suppliers (Milwaukee, Makita) for prize sponsorship

## 5. White Label Concept Document

### Definition
White-label AutoBookr allows other companies (partners, agencies, franchise groups) to rebrand and resell AutoBookr as their own product.

### Use Cases

| Partner Type | Example | Value to Them |
|--------------|---------|---------------|
| Franchise groups | Plumbing franchises | Branded booking system for all branches |
| Agencies | Marketing agencies | Add booking software to client services |
| Trade suppliers | Boiler manufacturers | "Book a service" for customers |
| Insurance | Home emergency cover | Claims + repair booking flow |
| Utilities | Energy suppliers | Installer scheduling |

### White Label Tiers

| Feature | Essential | Professional | Enterprise |
|---------|-----------|--------------|------------|
| Rebranding | Logo only | Logo + colors | Full custom domain |
| Support | Email | Email + chat | Dedicated CSM |
| Pricing control | No | Yes (markup) | Full control |
| Multi-tenant | - | 10 sub-accounts | Unlimited |
| API access | - | Read | Full |
| Custom features | - | - | Yes |
| **Price/mo** | **£199** | **£499** | **£999+** |

### Partner Economics (Professional Tier Example)

```
Partner pays AutoBookr: £499/mo
Partner charges clients: £50-150/mo per sub-account
10 clients = £500-1,500/mo revenue
                         = 0-200% margin
```

### Technical Requirements

1. **Branding**
   - Custom CSS/theme injection
   - Logo upload (light/dark variants)
   - Custom email sender (SPF/DKIM)

2. **Domain**
   - CNAME configuration
   - SSL managed

3. **Data**
   - Separate database per white-label
   - No cross-tenant data sharing

4. **Billing**
   - Partners bill their clients directly
   - AutoBookr invoices partner monthly

### Roadmap for White Label

| Phase | Timeline | Deliverable |
|-------|----------|-------------|
| Phase 1 | Q2 2026 | Logo + color rebranding (Essentials) |
| Phase 2 | Q3 2026 | Multi-tenant + custom pricing (Professional) |
| Phase 3 | Q4 2026 | Full API + custom features (Enterprise) |

### Risks & Mitigations

| Risk | Mitigation |
|------|------------|
| Brand damage by bad partners | Vetting process + SLA |
| Support overload | Tiered support model |
| Pricing undercutting | MAP (minimum advertised price) policy |
| Feature requests | Prioritised roadmap, extra dev = extra fees |

### First White Label Pilot
- **Target**: 1 plumbing franchise group (5-10 branches)
- **Goal**: Prove the model, refine onboarding
- **Timeline**: 3-month pilot starting Q2 2026
